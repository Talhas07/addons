FAQ

Does this module require a paid WhatsApp API or third-party service?

No. This module uses WhatsApp Web to send messages directly — no third-party API required.

Will the customer receive PDF documents?

Absolutely. PDFs such as quotations, invoices, receipts, and delivery orders can be attached manually and sent via WhatsApp.

Does it support multiple WhatsApp numbers or senders?

Yes, for each Odoo user you can use different WhatsApp number to login.

Are messages logged anywhere in Odoo?

Yes. Each message is logged in the document’s chatter, including delivery timestamp and status.

Is this module compatible with community and enterprise editions?

Yes, the module works with both Odoo 18 Community and Enterprise editions.

Is this module compatible with Odoo.sh?

No, this module is not compatible with Odoo.sh
